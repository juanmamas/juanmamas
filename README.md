# ¡Hola! Soy Mascotena Juan Manuel 

### Desarrollador Full Stack .NET 

Con más de 5 años de experiencia, me enfoco en construir sistemas sólidos y escalables utilizando tecnología .NET y Clean Architecture. Mi prioridad es escribir código mantenible y eficiente, aplicando principios SOLID para facilitar el crecimiento de los productos a largo plazo.

Tengo experiencia técnica en la implementación de Microservicios, gestión de tráfico con YARP API Gateway y estrategias de resiliencia con Polly. Además, diseño flujos de comunicación asíncrona mediante RabbitMQ y notificaciones en tiempo real con SignalR.

Complemento mi perfil backend con el desarrollo de interfaces modernas y dinámicas en React y React Native, buscando siempre un equilibrio entre la experiencia del usuario y el rendimiento del sistema.


---

### 🛠 Mi Stack Técnico
- **Backend:** .NET, ASP.NET Core, SignalR.
- **Arquitectura:** Clean Architecture, Microservicios, YARP API Gateway, CQRS.
- **Frontend/Mobile:** React, React Native, TypeScript.
- **Infraestructura:** PostgreSQL, MySQL, SQL Server, Firebase, Supabase, RabbitMQ, Docker.
